# Chaos-Agent.ai
Autonomous Swarm QA Testing Platform

## Monorepo Struktur
-  - Next.js Frontend
-  - Express Backend API
-  - Playwright Agent Worker
-  - Shared Types & Utilities

## Schneller Start

> chaos-agent@1.0.0 dev
> turbo run dev

## Deployment
- Frontend: Netlify/Vercel
- Backend: Railway/Render
- Agent Workers: Railway/Docker
